# Real-Time Chat Application

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io
- JWT Authentication

## Features
- User Login with JWT
- Real-time chat using Socket.io
- Multi-user chat support
- MongoDB message storage
- Logout functionality

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start

## Usage
- Open two browsers or tabs
- Login with different usernames
- Send messages in real time
